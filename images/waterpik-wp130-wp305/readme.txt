使用方法
1. 用喷头喷你的每一个牙缝，甚至牙龈和牙齿边缘。最里面的臼齿牙缝也能完全清洗到。
2. 建议早晚刷牙都要使用。
3. 一家人都可以使用。建议每人用一个自己的喷头。更换喷头只要按手柄上的EJECT按钮就轻松搞定。 
4. 使用自来水或加入适量漱口水来做水牙线清洁

注意事项
1. 这款水牙线是120V电压，需另购配备220V转120V或者110V的变压器和国标到美标的转换插头，国内应该很容易买到的（变压器和转换插头可以分开买，也可以买一体的）。亲们千万不能直接插到电源插座上，否则会烧毁哦。
2. 水压控制一定要从最低档开始用，然后根据自己的具体情况慢慢加大压力，个人的承受能力不一样。 可能要过几个月才能承受高压，或者你可以停留在你自己觉得舒适的压力档。我自己十几年前牙龈很不健康，牙医推荐用的。一开始我用低档冲，经常冲得牙龈出血。但是冲完以后觉得牙齿超爽。大概一个月以后才慢慢加大压力，而且再也没有出现牙龈出血的情况。现在用最高档也没有问题。
3. 如果小孩用的话一定要家长在旁边指导看护！ 防止小孩擅自更换到高压档，或者使用不当！！
